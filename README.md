# Advanced Timezone Converter Userscript

[![Download Latest](https://img.shields.io/github/v/release/Saxtus/advanced-timezone-converter-userscript?label=Download%20Latest&logo=github)](https://github.com/Saxtus/advanced-timezone-converter-userscript/releases/latest) ![GitHub Release Date](https://img.shields.io/github/release-date/Saxtus/advanced-timezone-converter-userscript) ![GitHub Downloads (all assets, all releases)](https://img.shields.io/github/downloads/Saxtus/advanced-timezone-converter-userscript/total)

A vibe-coded userscript that converts date/time strings on webpages from a known source timezone to your local timezone.

## Features
- Converts multiple date/time formats (2025-04-25 (16:17), ISO, US, EU, etc.)
- Configurable per-domain source timezone
- Supports dynamic web content

## Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/).
2. [Click here to install the script](https://github.com/Saxtus/advanced-timezone-converter-userscript/raw/refs/heads/main/timezone-converter.user.js)
3. Configure per-site timezone with the menu option.
